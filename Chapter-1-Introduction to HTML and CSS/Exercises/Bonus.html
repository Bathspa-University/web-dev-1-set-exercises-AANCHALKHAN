HTML & CSS Quick-Reference Guide

HTML Tags

  • h1-h6: h1,Defines headings whereas from H2-H6 reffered to sub headings
  • p: Defines a paragraph.
  • a: Defines a hyperlink, which is used to link from one page to another
  • img: Embeds an image in a web page
  • ul,ol,li: Creates unordered or ordered lists and list items.
  • div: a special element that lets you group similar sets of content together on a web page
  • span: Defines a inline container element.
  • table,tr,td: Creates a table and its rows and cells.
  • form,input: Builds interactive forms.
  • button: Defines a clickable elements that take actions.

CSS Properties

  • color: sets the currentcolor value.
  • font-family: specifies the font for an element.
  • background-color: Sets the background color of an element.
  • margin: specifies the space around an element's border.
  • padding: Defines the padding around an element's content.
  • border: Sets the border of an element.
  • display: Specifies the display behavior of an element.
  • width: Sets the width of an element.
